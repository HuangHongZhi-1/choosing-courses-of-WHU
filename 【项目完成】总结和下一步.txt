🎯 完整工作流程

【第1步】了解课程（5分钟）
  python whu_login.py
  → 输入学号和密码
  → 查看所有可选课程
  → 记下你想选的课程号

【第2步】添加课程到计划（5分钟）
  python grab_whu_interactive.py
  → 选择 1（添加课程）
  → 输入课程号、课程名、教师名
  → 重复添加多门课程
  → 选择 2（查看所有课程）确认

【第3步】实时监控（可选）
  → 选择 5（实时监控课程状态）
  → 每5秒检查一次课程状态
  → 按 Ctrl+C 停止

【第4步】执行选课（1分钟）
  → 选择 6（开始选课）
  → 输入学号和密码
  → 脚本自动选课
  → 显示结果统计

【第5步】查看结果（1分钟）
  → 选择 7（查看选课历史）
  → 看选课成功率和详细结果

总耗时：约15-20分钟（不含等待时间）
